title: My New Life
date: 2017-05-25 16:57:17
tags: git hexo创建博客的步骤

1.在GitHub创建新项目，名字有讲究，名字为:用户名.github.io

2.使用hexo框架搭建博客

首先全局安装hexo $ npm install hexo-cli -g

3.在电脑上建立博客的物理仓库 $ hexo init blog ;$ cd blog

4.在当前文件夹下执行初始化 $ hexo init

5.下载文件依赖包 $ npm install

为了将博客发到GitHub上，还需下载部署依赖包
$ npm install hexo-deployer-git –save

6.打开目录下的_config.yml文件，编辑里面的title：博客名

subtitle：副标题 description：描述 author：作者

url：http://用户名.github.io type: git repo:地址

该地址需要到github博客项目下，找到绿色按钮并点击，复制弹出框里的地址
save

7.生成博客所需文件 $ hexo g/generate 生成

8.本地试运行博客,启动服务器默认访问网址为：http://localhost:4000/
$ hexo s/server
关掉服务，依次运行下面的命令：
hexo g/generate 部署之前预先生成静态文件
hexo d/deploy 部署网站

9.新建博客 $ hexo new “博客标题”

备注：如果想要自己新建的每篇博客都有一个相应的图片文件夹，可以通过将 config.yml 文件中将 post_asset_folder 选项设为 true 来打开。

_config.yml
post_asset_folder: true

这样每次 $ hexo new “博客标题” 命令创建新文章时都会自动创建一个和博客标题同名的文件夹。将所有与你的文章有关的资源放在这个关联文件夹中之后，你可以通过相对路径来引用它们，这样你就得到了一个更简单而且方便得多的工作流。

10.在当前文件夹下的source文件夹下的_posts文件下找到新建博客文件，开始编辑

11.博客写完后，运行 $ hexo g 生成博客

12.可以运行 $ hexo s 试运行博客（本步骤可省）

13.然后将博客部署到github上 $ hexo d (第一次部署可能需要用户名和密码)
